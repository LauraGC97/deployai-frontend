Instalación y configuración

Deploy AI está compuesto por dos partes: el front-end en Angular y el back-end en Node.js. Ambos deben estar corriendo para que la IA funcione.

Requisitos previos

Node.js v18 o superior
Angular CLI v17 o superior
API Key de Groq console.groq.com (gratuito)

1. Clonar el repositorio

terminal 
# Clona el proyecto
git clone https://github.com/tu-usuario/deploy-ai
cd deploy-ai

2. Configurar el back-end

terminal 
cd deploy-backend
npm install
# Crea el archivo .env con tu API key
echo "GROQ_API_KEY=tu_api_key" > .env
npm run dev
ℹ️ El servidor arrancará en http://localhost:3000

3. Configurar el front-end

terminal 
cd DeployAi
npm install
ng serve
ℹ️ La aplicación estará disponible en http://localhost:4200

Modos de la IA

Deploy AI tiene tres modos especializados, cada uno con un system prompt optimizado para una tarea concreta.

⚡
deployFast() — Generar código
Genera código completo desde una descripción

Describe la funcionalidad que necesitas en lenguaje natural y la IA generará el código completo, limpio y comentado. Indica automáticamente el lenguaje usado y las dependencias necesarias.

Ejemplo de uso "Genera un servicio Angular que consuma una API REST con manejo de errores"
🔍
if (intelligent) — Analizar código
Detecta errores y sugiere mejoras

Pega tu código y la IA lo revisará en detalle: detectará bugs, malas prácticas, problemas de rendimiento y te sugerirá mejoras concretas con explicaciones.

Ejemplo de uso "Aquí mi componente Angular, ¿hay algún problema de rendimiento?"
📖
explain() — Explicar concepto
Tutorías de programación personalizadas

Pregunta cualquier concepto de programación y la IA lo explicará con ejemplos prácticos adaptados a tu nivel. Ideal para aprender nuevas tecnologías.

Ejemplo de uso "¿Cuál es la diferencia entre Observable y Promise en Angular?"

Cómo usar el chat

La interfaz del chat está diseñada para ser intuitiva. Aquí tienes todo lo que necesitas saber.

Interfaz principal

1
Panel lateral — Modos y sugerencias

Selecciona el modo según tu necesidad. Las sugerencias rápidas te permiten empezar con un clic sin escribir nada.

2
Área de mensajes

Las respuestas de la IA se renderizan con formato Markdown — negritas, bloques de código con syntax highlight, listas y tablas.

3
Input de texto

Escribe tu mensaje y pulsa Enter para enviar. Usa Shift + Enter para nueva línea sin enviar.

4
Historial de conversaciones

Tus conversaciones se guardan automáticamente en el navegador. Puedes recuperarlas desde el panel lateral en cualquier momento.

Consejos para mejores resultados

→ Sé específico en tu descripción — cuanto más detalle, mejor código genera.
→ Para analizar código, pega el código completo para un análisis más preciso.
→ Puedes hacer preguntas de seguimiento — la IA recuerda el contexto de la conversación.
→ Indica el lenguaje o framework que usas para obtener código más preciso.

Referencia de la API

El back-end expone una API REST en http://localhost:3000. Todos los endpoints devuelven JSON.

Health check

GET /health

Verifica que el servidor está funcionando correctamente.

respuesta
"status": "ok",
"service": "Deploy AI Backend"

Enviar mensaje

POST /api/chat

Envía un array de mensajes al modelo Llama 3 de Groq y devuelve la respuesta.

Body (JSON)
request
"messages": [
  {
    "role": "user",
    "content": "Genera un componente Angular"
  }
]
Respuesta exitosa — 200
response
"content": "Aquí tienes el componente..."
Respuesta de error — 400 / 500
response
"error": "El campo messages es obligatorio"
Parámetros del mensaje
Campo Tipo Valores Descripción
role string user · assistant Quién envía el mensaje
content string cualquier texto Contenido del mensaje

Modelo utilizado

llama-3.3-70b-versatile
Groq 70B parámetros Gratuito ~400 tokens/s

Modelo Llama 3.3 de Meta ejecutado en la infraestructura de Groq. Ofrece velocidad de inferencia muy alta con un límite de 14.400 requests/día en el tier gratuito.
